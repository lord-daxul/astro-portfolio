# daxulfolio

Portfolio headless: **WordPress** (WPGraphQL) + **Astro**, pensado para **Cloudflare Pages**.

## Desarrollo

```bash
npm install
npm run dev
```

Copia `.env.example` a `.env` y define al menos `PUBLIC_WORDPRESS_GRAPHQL_URL`.

## Build

```bash
npm run build
```

Variables para producción (p. ej. Cloudflare Pages): ver `.env.example`.
